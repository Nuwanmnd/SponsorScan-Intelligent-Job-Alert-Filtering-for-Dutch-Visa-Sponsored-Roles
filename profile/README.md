# Profile Files

These files are used by the AI review feature in the React app.

Files:

- `cv.md`: your current CV in text form
- `cover_letter.md`: your base cover letter or motivation letter
- `courses.md`: courses, certifications, and training
- `extra_notes.md`: anything else you want included in the prompt context

Setup:

1. Copy `.env.example` to `.env`
2. Put your Anthropic API key in `.env`
3. Fill in the files in this folder with your real content

You can also edit the prompt behavior in the `prompts` folder.
